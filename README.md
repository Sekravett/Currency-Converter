# Currency Converter - Конвертер валют

**Автор:** Маликов Илья

## Описание
GUI-приложение для конвертации валют с использованием актуальных курсов валют через API exchangerate-api.com. Приложение сохраняет историю конвертаций и позволяет экспортировать её в JSON формат.

## Возможности
- 🔄 Конвертация между 15+ валютами
- 📊 Актуальные курсы валют через API
- 📝 Сохранение истории конвертаций в JSON
- 💾 Экспорт истории в отдельный файл
- 🎨 Удобный графический интерфейс
- ⚡ Автономный режим с демо-курсами
- ✅ Проверка корректности ввода

## Требования
- Python 3.7 или выше
- Установленные библиотеки из requirements.txt
- Интернет-соединение для получения курсов валют
